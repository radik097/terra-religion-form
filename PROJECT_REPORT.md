# PROJECT_REPORT

## Проект

`Terra Srezannaya — Religion Package Builder`

## Назначение

Создать GitHub Pages форму для оформления новой религии в лоре и скачивания минимального ZIP-пакета файлов.

## Реализовано

- Статический сайт без backend.
- Форма создания религии.
- Простые подсказки на каждом этапе.
- Загрузка фото значка на карте.
- Загрузка фото города.
- Сохранение записей в `localStorage`.
- Реестр текущих религий.
- Поиск и фильтр по статусу.
- Экспорт/импорт локального реестра JSON.
- Генерация ZIP в браузере без внешних библиотек.
- Генерация `index.json` внутри ZIP.
- Генерация `RELIGION.md` внутри ZIP.

## Минимальный пакет религии

```text
<religion-slug>/
├─ index.json
├─ RELIGION.md
└─ assets/
   ├─ map_icon.<ext>
   └─ city_photo.<ext>
```

## Входные поля

```text
name
сuthor / author
basis
description
rulers
status
tag
mapIcon
cityPhoto
```

## Техническая архитектура

```text
index.html   — структура страницы и форма
styles.css   — визуальный стиль, адаптивность, карточки
app.js       — состояние, localStorage, ZIP builder, экспорт/импорт
data/        — документация схемы seed/schema
.github/     — GitHub Pages workflow
```

## Ограничения

- Нет общего серверного списка пользователей.
- Данные сохраняются только в браузере пользователя.
- Большие изображения могут переполнить `localStorage`.
- ZIP создаётся на стороне клиента.
